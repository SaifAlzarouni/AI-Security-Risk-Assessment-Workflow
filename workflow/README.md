# n8n Workflow

This folder contains the exported n8n workflow used for the AI Security Risk Assessment Workflow project.

The workflow automates the following process:

1. Accept meeting notes or a PDF document.
2. Extract text from uploaded PDF files.
3. Generate an executive summary using Google Gemini.
4. Analyze the summary for cybersecurity risks.
5. Classify and prioritize the most critical risks.
6. Generate a professional Gmail draft ready for review.

The workflow JSON file can be imported directly into n8n.
